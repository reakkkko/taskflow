# Taskflow 📝

A full stack todo app with authentication — built with React, TypeScript, Tailwind CSS, Node.js, Express and PostgreSQL.

## Features

- Register & Login with JWT authentication
- Personal todos — every user sees only their own tasks
- Add, complete and delete todos
- Progress bar
- Protected routes
- Passwords hashed with bcrypt

## Tech Stack

**Frontend** — React + TypeScript + Vite + Tailwind CSS

**Backend** — Node.js + Express + PostgreSQL + JWT

→ Backend repo: [taskflow-backend](https://github.com/reakkkko/taskflow-backend)

## Getting Started

### Prerequisites

- Node.js
- Backend running on `http://localhost:3001`

### Installation

```bash
git clone https://github.com/reakkkko/taskflow
cd taskflow
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
